 vrunner help
vanessa-runner v2.5.0
Возможные команды:
 help             -      Вывод справки по параметрам.
 --help           -      Вывод справки по параметрам.
 version          -      Вывод версии приложения.
 init-project     -      Создание структуры проекта.
 init-dev         -      Инициализация базы данных для выполнения необходимых тестов.
 update-dev       -      Обновление ИБ 1С.

 xunit            -      Запуск тестирования через фреймворк Vanessa-ADD (Vanessa Automation Driven Development).
 vanessa          -      Запуск проверки поведения (BDD) через фреймворк Vanessa-ADD (Vanessa Automation Driven Development).
 run              -      Выполнение команды/действия в 1С:Предприятие в режиме тонкого/толстого клиента с передачей запускаемых обработок и параметров.
 designer         -      Запуск Конфигуратора 1С - интерактивно или автоматически.

 compileext       -      Загрузка расширения в конфигурацию из папки исходников.
 loadext          -      Загрузка расширения в конфигурацию из cfe-файла.
 updateext        -      Обновление расширения в ИБ.
 compileexttocfe  -      Сборка cfe-файла из исходников.
 decompileext     -      Разборка расширения из конфигурации в исходники.
 unloadext        -      Выгрузка файла расширения из ИБ.

 compileepf       -      Сборка внешних обработок из исходников штатно через выгрузку 1С 8.3.
 decompileepf     -      Разборка внешних обработок на исходники штатно через выгрузку 1С 8.3.

 updatedb         -      Обновление конфигурации базы данных в режиме Конфигуратора.
 unload           -      Выгрузка файла конфигурации из ИБ.
 make-dist        -      Выгрузка файла поставки конфигурации из ИБ.
 dump             -      Выгрузка информационной базы в dt-файл.
 restore          -      Загрузка информационной базы из dt-файла.
 create           -      Создать информационную базу через ras\rac.
 remove           -      Удалить информационную базу через ras\rac.

 loadrepo         -      Обновление конфигурации из хранилища 1С. Обновление БД не выполняется.
 createrepo       -      Создание хранилища 1С.
 bindrepo         -      Подключение ИБ к хранилищу конфигурации 1С.
 createrepouser   -      Создание пользователя хранилища конфигурации 1С.
 copyrepouser     -      Копирование пользователей хранилища из другого хранилища.

 unloadcfrepo     -      Выгрузка файла конфигурации определенной версии из хранилища 1С.
 lockrepo         -      Захват объектов в хранилище 1С.
 commit           -      Помещение изменений объектов в хранилище конфигурации.
 unbindrepo       -      Отключение ИБ от хранилища конфигурации 1С.

 unlockrepo       -      Отмена захвата объектов в хранилище 1С.
 compile          -      Сборка cf-файла из исходников.
 decompile        -      Разборка конфигурации в исходники.
 load             -      Загрузка cf-файла в базу.
 merge            -      Объединение cf-файла c конфигурацией из инф.базы.
 update           -      Обновление конфигурации, находящейся на поддержке, или объединение конфигурации с файлом (с использованием файла настроек).
 compare          -      Формирование файла отчета сравнения конфигураций.

 session          -      Управление сеансами информационной базы.
 scheduledjobs    -      Управление возможностью работы регламентных заданий.
 dbinfo           -      Получение информации о базе данных (выводится в консоль выполнения скрипта).
                Может применяться для проверки работы RAS/RAC.
 syntax-check     -      Централизованная проверка конфигурации, в т.ч. полная проверка синтаксиса конфигурации.
 edt-validate     -      Проверка проекта EDT.
 set-version      -      Установка номера версии файлов 1С (конфигурации, расширения, внешние обработки\отчеты).
